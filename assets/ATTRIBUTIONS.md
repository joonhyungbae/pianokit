## 곡 선택 이유

두 곡은 **시각적 대비를 극대화하기 위해** 선택되었습니다:

| | Satie Gymnopédie No.1 | Prokofiev Toccata Op.11 |
|---|---|---|
| 템포 | 느림 (~72 BPM) | 빠름 (~160+ BPM) |
| 텍스처 | 얇음 (1-3음) | 두꺼움 (6-10음) |
| 음역 | 좁음 (C3-C5) | 전체 건반 (A0-C8) |
| 다이내믹 | pp-mp | f-fff |
| 어택 | 레가토, 희소 | 타격적, 밀집 |
| 루바토 | 거의 없음 | 정확한 펄스 |

이 대비는 pianokit의 cue 파라미터(energy, density, register, attack, harmonic_tension)를 통해 **질적으로 다른 시각 세계**로 표현됩니다.
